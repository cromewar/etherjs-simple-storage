# EtherJs Simple Storage

Simple Project deploying SimpleStorage contract using Ethers JS.

### Steps to Follow

1. Rename the `env.example` file to `.env`.
2. Replace the examples with your own RPC_URL and PRIVATE_KEY.
3. run the code with `node deploy.js`.

### Additional

There is a script to encrypt de private key in order to don't use plain text or .env file.
